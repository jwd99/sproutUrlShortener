URL SHORTENER

Make API calls to shorten urls.

API REQUEST EXAMPLES

POST: /shorten
BODY: {"url": "https://www.google.com"}
RESPONSE: {originalUrl:"https://www.google.com", shortUrl: "https://localhost:8000/r309jn"}
-NOTE BODY URL must be a url that returns a valid 200 response. localhost may be on 3000 for others


GET: /'shorten url goes here'
RESPONSE: redirect to original url
